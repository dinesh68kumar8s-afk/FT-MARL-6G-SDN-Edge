## Description
This repository contains the implementation of the FT-MARL framework used for:

- Federated multi-agent reinforcement learning
- Trust-aware aggregation
- Semantic-aware communication modeling
- Carbon-aware resource scheduling
- Simulation of 6G SDN–Edge IoT networks
- Generation of Figures and Tables reported in the manuscript

The code reproduces simulation results, figures, and metrics used in the paper.
